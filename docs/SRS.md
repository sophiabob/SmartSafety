# SRS — Software Requirements Specification
## Система видеоаналитики для контроля техники безопасности "SafetyVision"
## 1. Введение
### 1.1. Назначение документа
Данный документ содержит полный перечень функциональных и нефункциональных требований к системе видеоаналитики "SafetyVision", предназначенной для автоматического контроля соблюдения техники безопасности на промышленных объектах.
### 1.2. Область применения
Система предназначена для использования на промышленных объектах (заводы, стройплощадки, производственные цеха) с целью автоматического контроля:
- наличия защитной каски у сотрудников;
- наличия сигнального жилета у сотрудников;
- нахождения сотрудников в опасных зонах.
### 1.3. Целевая аудитория
- Операторы систем видеонаблюдения
- Специалисты по охране труда
- Руководители промышленных объектов
## 2. Общее описание системы
### 2.1. Перспективы продукта
Система представляет собой программное обеспечение для обработки видеофайлов с целью выявления нарушений техники безопасности. Входными данными являются видеофайлы с камер наблюдения. Выходными данными — обработанное видео с визуальной разметкой и структурированный отчет о нарушениях.
### 2.2. Пользовательские характеристики
- **Оператор** — загружает видео, запускает анализ, просматривает результаты
- **Инженер по ТБ** — анализирует отчеты о нарушениях, принимает меры
- **Руководитель** — получает сводную статистику по нарушениям
### 2.3. Ограничения и допущения
| Ограничение | Описание |
|-------------|----------|
| Входные данные | Только готовые видеофайлы (MP4, AVI), без работы с IP-камерами |
| Оборудование | Только ПК, без физических камер |
| Вычисления | CPU (GPU опционально) |
| Модель AI | Предобученная YOLOv8 (без дообучения) |
| Интерфейс | PyQt5/PySide6 |
